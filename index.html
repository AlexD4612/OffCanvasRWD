Responsive for the Holidays

Welcome to Responsive for the Holidays! This site demonstrates responsive web design and makes use of a top down off-canvas menu.

Different devices will see different holidays and respective information about it! For example, you are currently on Desktop so you'll see everything about Halloween.

To learn more about which holidays are visible on which devices, check out the media queries tab!
